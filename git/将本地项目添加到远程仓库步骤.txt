一 Create a new repository on the command line

git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/sawakonotsadako/spring-boot-learning.git
git push -u origin master

git remote add origin https://github.com/sawakonotsadako/spring-boot-learning.git
git push -u origin master
